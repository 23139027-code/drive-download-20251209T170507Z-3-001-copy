Cấu hình MQTT Broker

Cấu hình kết nối MQTT để giao tiếp với ESP32. Sau khi lưu, trang web sẽ tải lại để áp dụng.

Ví dụ: broker.emqx.io, mqtt.eclipse.org
Port WebSocket (thường là 8083 hoặc 8084)
Đường dẫn WebSocket (thường là /mqtt)
Username để xác thực MQTT (nếu broker yêu cầu)
Password để xác thực MQTT (nếu broker yêu cầu)
Thời gian duy trì kết nối (mặc định 60s)

Thông tin Topic

Gửi lệnh đến ESP32: DATALOGGER/{deviceId}/CMD

Nhận dữ liệu từ ESP32: DATALOGGER/{deviceId}/DATA

Xem file ESP32_MQTT_GUIDE.md để biết cách cấu hình ESP32

Cấu hình WiFi cho ESP32 lần đầu

Nếu ESP32 chưa kết nối WiFi, bạn cần cấu hình WiFi trước khi sử dụng hệ thống.
